# Survey on Driverless Vehicles

## Overview
This repository contains a group presentation exploring **Autonomous Vehicles (AVs)**, commonly known as self-driving vehicles. The presentation explains how autonomous vehicles operate, the different levels of automation, the technologies that enable autonomous driving, current challenges, infrastructure requirements, and the future impact of AVs on society.

The presentation was delivered orally to an audience of approximately **20 people**, with the slides serving as visual aids to support discussion and explain complex technical concepts.

---

## My Contribution

I was responsible for creating and presenting **Slides 11–17**, which covered the technology behind autonomous vehicles. My section focused on explaining the hardware and networking systems that allow autonomous vehicles to perceive their environment, determine their position, and communicate with onboard systems.

Topics included:

- **Autonomous Vehicle Technologies**
  - LiDAR
  - Cameras
  - GPS
  - Radar
  - Sensor arrays
  - Vehicle control algorithms

- **Sensor Functions**
  - LiDAR (Light Detection and Ranging)
  - Visible Light Communication (VLC)
  - Ultrasonic Ranging Devices (URD)
  - Millimeter Wave Radar
  - Infrared Ranging

- **Positioning Technologies**
  - GPS
  - Doppler Radar Speedometers
  - Radar Cruise Control
  - Radar-Based Obstacle Detection

- **Vision Systems**
  - Camera placement and object detection
  - Traffic sign recognition
  - Lane detection
  - Blind spot elimination

- **Vehicle Networking**
  - Vehicular Ad-Hoc Networks (VANETs)
  - Automotive Ethernet
  - Controller Area Network (CAN)
  - FlexRay
  - Local Interconnect Network (LIN)
  - Byteflight
  - Low-Voltage Differential Signaling (LVDS)
  - Media Oriented Systems Transport (MOST)
